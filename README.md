# Convertidor DC/DC Web App

Aplicación web con:

- FastAPI (backend)
- DynamoDB (base de datos)
- Autenticación JWT
- Frontend HTML/CSS/JS

## Funcionalidades

- Registro de usuarios
- Login con token
- Simulación de convertidor DC/DC

## Tecnologías

- Python
- FastAPI
- AWS DynamoDB
- JavaScript
